## Additional Resources

### Full Testing Guide
For detailed instructions on testing each feature, see BETA_TESTING_GUIDE.md

### Document Recommendations
For best results, test with:
- **Short document** (1-3 pages) - Quick testing
- **Medium document** (5-10 pages) - Standard use case
- **Long document** (20+ pages) - Stress testing

Avoid:
- Scanned PDFs (images only - OCR not supported yet)
- Password-protected/encrypted PDFs
- Very large files (100MB+ may timeout)

### Common Issues & Solutions

**"Failed to upload document"**
- Check file size (max 100MB) and format (PDF, DOCX, DOC, TXT only)
- Try a different file

**"Mind map generation failed"**
- Large documents may take 2-5 minutes
- Try a smaller document or refresh and retry

**"Flashcard not found" error**
- This is expected for old flashcards - regenerate to enable progress tracking

**Can't sign in**
- Clear browser cookies and try again
- Make sure you verified your email
