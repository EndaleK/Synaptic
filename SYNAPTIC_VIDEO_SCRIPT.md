## TECHNICAL SETUP

### Recording Settings (Screen Capture)
- **Resolution**: 1920x1080 minimum (4K preferred)
- **Frame Rate**: 60fps for smooth UI interactions
- **Cursor**: Use cursor highlighting/ring effect
- **Audio**: Record voiceover separately for better quality
- **Format**: H.264 (MP4) for compatibility

### Editing Software Recommendations
- **Professional**: Adobe Premiere Pro, Final Cut Pro
- **Mid-tier**: DaVinci Resolve (free version is excellent)
- **Quick**: Camtasia (screen recording + editing)
- **Online**: Descript (AI-powered, great for voiceovers)

### Export Settings
- **YouTube**: H.264, 1080p, VBR 2-pass, 8-10 Mbps
- **Social**: Square (1:1) or Vertical (9:16) crops
- **Captions**: Include SRT file for accessibility
