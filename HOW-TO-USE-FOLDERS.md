## Adding Documents to a Folder

### Method 1: Drag and Drop
1. Click and hold a document card
2. Drag it over the folder you want
3. Release to drop it into the folder

### Method 2: Move via Menu

## Opening a Folder

1. Click on any folder card in your library
2. The folder opens showing all documents inside
3. You'll see the folder name at the top with a breadcrumb trail

---

## Navigating Between Folders

**Breadcrumb Navigation:**
- At the top of the library, you'll see: `Home > Biology 101 > Lab Reports`
- Click any part of the breadcrumb to jump back to that level

**Back Button:**
- Click the back arrow to go up one level

## Searching Within Folders

1. Open a folder
2. Use the search bar at the top
3. Search will filter documents within the current folder only

**Tip**: To search all documents (including those in folders), search from the main library view.

## Troubleshooting

**Q: I can't find my document after moving it to a folder**
- Open the folder you moved it to
- Check the breadcrumb trail to see your current location
- Use the search function to find the document

**Q: My folder disappeared**
- Check if you accidentally deleted it
- Look in your main library - deleted folders move documents back to root

**Q: Can I move multiple documents at once?**
- Currently, you need to move documents one at a time
- Multi-select is coming in a future update

**Q: What happens if I upload a document while inside a folder?**
- The document will be added to the current folder you're viewing
